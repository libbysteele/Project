<h1>Explanation of My Python Calendar Program</h1>


<p>This HTML file explains the functionality of my python program for creating a calendar with key holidays and additional features.It also reviews my program.</p>


<h2>Key Features:</h2>


<ul>

    <li><strong>Display Calendar:</strong> The program displays a calendar for a chosen month and year of the user.</li>

    <li><strong>Show Key Holidays:</strong> Key holidays for the selected month are highlighted and listed in the calendar.</li>

    <li><strong>Add Custom Holidays:</strong> Users can add their own custom holidays to the calendar.</li>

    <li><strong>View Other Months and Years:</strong> Users have the option to view calendars for other months and years.</li>

</ul>


<h2>How It Works:</h2>


<p>The Python program utilises a calendar library to generate the calendar for the specified month and year. It also incorporates a list of key holidays, which are highlighted and listed within the calendar.</p>


<p>Users can interact with the program through a user-friendly interface, where they input the desired month and year. Additionally, they have the option to add custom holidays by providing the holiday name and date.</p>


<p>Upon request, the program displays the generated calendar with key holidays highlighted. Users can navigate through different months and years to view calendars for specific dates.</p>


<h2>Content:</h2>


<p>My custom holidays relies on mathematical algorithms for example the calculation I used for Easter Sunday is the Butcher algorithm which I found through research.</p>


<p>I have used multiple conditional loops with error-handling incorporated to show technical skills.</p>
